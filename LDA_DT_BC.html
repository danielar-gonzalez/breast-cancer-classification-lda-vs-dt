Este código realiza la clasificación de tumores benignos y malignos utilizando el conjunto de datos Breast Cancer de scikit-learn. Primero, cargamos el conjunto de datos y lo preprocesamos para separar las características predictoras (X) y la variable objetivo (y). Luego, dividimos los datos en conjuntos de entrenamiento y prueba, asegurando que la distribución de las clases (benigno y maligno) se mantenga equilibrada en ambas particiones mediante el parámetro stratify=y de train_test_split. Esto es importante para evitar sesgos en el entrenamiento y evaluación del modelo, especialmente cuando se trabaja con clases desbalanceadas.

from sklearn.metrics import accuracy_score, f1_score

tree = DTC(random_state=42)
tree.fit(X_train, y_train)

plot_tree(tree, filled=True, feature_names=variables_importantes, class_names=['Benigno', 'Maligno'])

y_pred = tree.predict(X_test_scaled)
acc0 = accuracy_score(y_test, y_pred)
f10 = f1_score(y_test, y_pred)

print("F1-score inicial:", f10)

# Validación cruzada para encontrar el mejor valor de alpha
skf = StratifiedKFold(n_splits=4)
ccp_values = np.linspace(0.001, 0.2, 250)
cv_scores = []

for alpha in ccp_values:
    pruned_tree = DTC(ccp_alpha=alpha, random_state=42)
    cv_scores.append(np.mean(cross_val_score(pruned_tree, X_train_balanced, y_train_balanced, cv=skf, scoring='f1')))

best_alpha = ccp_values[np.argmax(cv_scores)]
print("Mejor valor de alpha:", best_alpha)

pruned_tree = DTC(ccp_alpha=best_alpha, random_state=42)
model_ptree = pruned_tree.fit(X_train_balanced, y_train_balanced)
plt.figure(figsize=(15, 10))
